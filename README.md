PHP Twitch API Wrapper for Laravel 5+

## Installation

Run

```
composer require romanzipp/twitch
```

Or add `romanzipp/twitch` to your `composer.json`

```
"romanzipp/twitch": "~1.0"
```

Run composer update to pull down the latest version.
